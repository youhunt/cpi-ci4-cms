# CPI Director Website Concept Alternative

Prototype HTML statis untuk bahan meeting direktur PT. Chugoku Paints Indonesia.

## Lokasi file

```text
public/demo/cpi-director-concepts/index.html
```

## Cara akses lokal

Jalankan aplikasi CodeIgniter 4:

```bash
php spark serve
```

Buka:

```text
http://localhost:8080/demo/cpi-director-concepts/index.html
```

## Fitur demo

- 3 konsep website: Corporate Identity, Product Focused, Service Focused
- Toggle bahasa ID/EN untuk headline utama
- Live homepage simulation
- Popup detail produk
- Filter produk
- Map pin clickable
- Inquiry form demo
- FAQ accordion

## Catatan

File ini adalah alternatif visual untuk presentasi. Tidak mengubah flow utama aplikasi CI4/CMS. Setelah konsep disetujui, desain ini bisa dipecah ke layout CI4, view partial, CSS, JavaScript, dan modul CMS bilingual.
